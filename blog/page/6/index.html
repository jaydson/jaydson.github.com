Esfera Com CSS3 3D

Mar 9th, 2012 | Comments

Fiz mais um demo com CSS3 para o Mozilla Dev Derby.
Depois de ter feito um cubo 3d, minha ideia agora foi complicar um pouco e fazer uma esfera.
Depois de pensar um pouco, saiu uma esfera legal.
Não desenvolvi a esfera com base nas fórmulas da geometria, o que seria o correto e faria ficar bem mais bacana, mas para um primeiro experimento, está Ok.

Root3D - Experimento CSS3D

Mar 8th, 2012 | Comments

Trata-se de um experimento, e digo experimento porque isso ainda é muito novo, onde é possível criar elementos 3D em um ambiente todo desenvolvido com CSS3.
Não, eu não falei besteira, é em CSS3 mesmo.
Só vendo para ter a real noção do que realmente é.

Cubo 3D Com CSS3

Mar 7th, 2012 | Comments

Neste mês de Março, o Mozilla Dev Derby tem como desafio desenvolver demos de CSS3 3D.
Pra quem não conhece o Dev Derby, é uma competição promovida pela Mozilla para que desenvolvedores do mundo inteiro façam aplicações com novas tecnologias.
Já participei, fazendo um experimento com o History API do HTML5:
https://developer.mozilla.org/en-US/demos/detail/history-image-gallery

Depois de assistir a palestra do @felipenmoura e de ter umas aulas com ele sobre CSS3, resolvi fazer um pequeno experimento para submeter para o DevDerby.
O experimento é um cubo 3D, totalmente feito com CSS3:
https://developer.mozilla.org/en-US/demos/detail/simple-css3-3d-cube

Confiram lá, e se curtirem, cliquem no “Like” ;)
