Experimento De Web Scraping Com nodeJS

Jun 30th, 2012 | Comments

Depois que fiz o post Web Scraping para crianças com nodeJS, cheguei a conclusão de que deveria fazer um experimento com o que mostrei.
Sentei algumas horas no final de semana, e pronto.
Basicamente fiz o que mostrei no outro post, porém fiz uma interface e fiz deploy da aplicação no NodeJitsu.
Utilizei outras coisas como o ExpressJS para criar a casca da aplicação e o Jade para template.
O código está disponível no Github >>> https://github.com/jaydson/myscraper
Para quem quiser testar online, a URL do MyScraper é esta: http://myscraper.jit.su/
